# Little Aprendiz Website

GitHub/Netlify-ready export from Readdy.

## Local preview

```bash
npm install
npm run dev
```

## Netlify settings

- Build command: `npm run build`
- Publish directory: `out`

## Images

All website images are stored in `public/assets/`.
